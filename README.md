>>> zip(letters, numbers)

# EXTREMLY IMPORTANT - ACCESSING DICT ELEMENTS IS CRUCIAL

## Update `ml_runner/__init__.py`

```python
from flask_debugtoolbar import DebugToolbarExtension
toolbar = DebugToolbarExtension(app)
```
